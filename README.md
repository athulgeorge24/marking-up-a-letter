# University Application Response Letter

A properly structured HTML document that marks up a university response letter to a PhD application.

## Features

- Semantic HTML5 elements (`<address>`, `<time>`, `<abbr>`, `<dl>`, lists)
- Professional letter format with sender and recipient information
- Text formatting with `<strong>`, `<em>`, subscripts, and superscripts
- Minimal CSS styling for clean presentation
- Responsive design

## Contents

The letter includes:
- Sender and recipient addresses
- Multiple sections covering starting dates, subjects of study, and exotic dances
- Definition lists for detailed information
- Proper semantic markup for accessibility
